# EducationalWeb
یک وب‌سایت چندصفحه‌ای برای یک آموزشگاه طراحی شده با استفاده از HTML، CSS و Bootstrap.در این وب‌سایت صفحات خانه، معرفی دوره‌ها، دپارتمان‌ها، جزئیات دوره، آخرین اخبار، ورود کاربران و کلاس‌های در حال برگزاری پیاده‌سازی شده‌اند.   طراحی سایت کاملاً واکنش‌گرا (Responsive) بوده و با هدف تمرین طراحی رابط کاربری مدرن و ساختاردهی صفحات آموزشی انجام شده است.
